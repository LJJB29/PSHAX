How to Install PS3HEN

Follow these steps to install HFW and PS3HEN on your PS3:

Step 1: Install HFW (Hybrid Firmware)

  1. Download the latest **HFW (Hybrid Firmware)** from a trusted source.
  2. Format a USB drive to **FAT32** and create a folder named **PS3**.
  3. Inside the **PS3** folder, create another folder named **UPDATE**.
  4. Place the downloaded **HFW update file** (renamed to **PS3UPDAT.PUP**) inside the **UPDATE** folder.
  5. Insert the USB into your PS3 and go to **System Update** > **Update via Storage Media**.
  6. Follow the on-screen instructions to install HFW.

Step 2: Install PS3HEN

  1. Once HFW is installed, go to ps3xploit.me on your PS3 browser.
  2. Navigate to the **PS3HEN Installer** section.
  3. Follow the instructions to install and activate PS3HEN.
  4. Reboot your console and reactivate HEN after each restart.
  5. Enjoy homebrew apps like MultiMan and WebMan.
